White Phoenix uses the following open source component:

python-docx: Python library for creating and updating Microsoft Word (.docx) files.
(https://github.com/python-openxml/python-docx)

NumPy: The fundamental package for scientific computing with Python. (https://github.com/numpy/numpy)

Pillow: The Python Imaging Library by Fredrik Lundh and Contributors. (https://github.com/python-pillow/Pillow)

PyPdf: A free and open-source pure-python PDF library capable of splitting, merging, cropping,
 and transforming the pages of PDF files. It can also add custom data, viewing options,
 and passwords to PDF files. pypdf can retrieve text and metadata from PDFs as well. (https://github.com/py-pdf/pypdf)
